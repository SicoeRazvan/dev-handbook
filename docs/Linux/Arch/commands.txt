### System Information ###
uname -a -> Show kernel version and system info
hostnamectl -> Show hostname and OS details
lsb_release -a -> Display Linux distribution info (requires lsb-release)
fastfetch -> Pretty system summary (if installed)


### Package Management (Pacman) ###
sudo pacman -Syu         -> Update system (sync packages + upgrade)
sudo pacman -S pkgname    -> Install a package
sudo pacman -R pkgname    -> Remove a package
sudo pacman -Ss term  -> Search for a package
pacman -Qs term       -> Search installed packages
pacman -Qi pkgname        -> Get info about a package
sudo pacman -U filename    -> Install package from local file


### File and Directory Management ###
ls -l                    -> List files in long format
cd /path/to/dir          -> Change directory
mkdir newdir             -> Create directory
rm -rf dir_or_file       -> Delete files/directories
cp file1 file2           -> Copy file
mv file1 file2           -> Move/rename file
find / -name file.txt    -> Search for a file


### Networking ###
ip a                     -> Show network interfaces
ping archlinux.org       -> Test network connectivity
sudo systemctl restart NetworkManager  -> Restart network


### Disk and Storage ###
df -h                    -> Show disk space usage
du -sh                 -> Show directory sizes
lsblk                    -> List block devices
mount /dev/sdX /mnt      -> Mount a drive
umount /mnt**              -> Unmount a drive


### Maintenance ###
sudo pacman -Rns $(pacman -Qdtq)   -> Remove orphaned packages
sudo journalctl -p 3 -xb            -> Show errors from last boot
sudo pacman -Sc                     -> Clear package cache
